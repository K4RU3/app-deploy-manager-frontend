# フロントエンド開発仕様書

（LLM主導開発用ドキュメント）

---

# 1. プロジェクト概要

本プロジェクトは **自宅サーバー用PaaS管理システム**のフロントエンドを開発する。

目的は以下。

* サービスの公開・停止管理
* Gitベースデプロイ管理
* Dockerコンテナ管理
* データボリュームバックアップ管理
* デプロイ履歴・ログの可視化

フロントエンドは **管理UI**として機能し、バックエンドAPIを操作する。

# 4. データモデル

## Service

```
Service {
  id: string
  name: string
  domain: string
  repo: string
  branch: string

  deployMode: "branch" | "commit"

  selectedCommit?: string

  port?: {
    host: number
    container: number
    protocol: "tcp" | "udp"
  }

  enabled: boolean

  dockerVolume: string

  autoBackup: boolean
}
```

---

## Commit

```
Commit {
  hash: string
  shortHash: string
  message: string
  author: string
  date: string
}
```

---

## Backup

```
Backup {
  file: string
  size: string
  createdAt: string
}
```

---

## DeployLog

```
DeployLog {
  id: string
  serviceId: string
  commit: string
  status: string
  createdAt: string
}
```
